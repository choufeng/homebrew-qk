# Choufeng Qk

## How do I install these formulae?

`brew install choufeng/qk/<formula>`

Or `brew tap choufeng/qk` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "choufeng/qk"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
